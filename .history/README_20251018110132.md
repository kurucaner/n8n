# n8n Docker Setup

## Quick Commands

```bash
# Start n8n
docker-compose up -d

# Stop n8n
docker-compose down

# Restart n8n
docker-compose restart

# View logs
docker-compose logs -f

# Apply changes (after updating docker-compose.yml or .env)
docker-compose down && docker-compose up -d
```

## Access

- n8n UI: http://localhost:5678
